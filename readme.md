# totalizador-de-resumo-mensal-de-saaj

Serviço online gratuito que auxilia na elaboração, como o nome sugere, do resumo mensal de SAAJ (Serviço Amigo do Adolescente e Jovem), por meio do cálculo automático dos totais com base nos dados preenchidos pelo usuário. Foi criado de acordo com o modelo da ficha de resumo mensal de SAAJ actualmente vigente no Serviço Nacional de Saúde em Moçambique.
